Descrição do Sistema:
Projeto desenvolvido em linguagem C para simular um sistema de Caixa Eletrônico (ATM), utilizando estruturas de repetição, decisão e funções.
O sistema permite ao usuário consultar saldo, realizar saques, depósitos e sair do programa através de um menu interativo no console.

Funcionalidades Implementadas:
Consultar saldo
Realizar saque
Realizar depósito
Menu interativo
Validação de saldo e valores inválidos
Compilação e Execução

Commitis foram feitos apenas por dois integrantes devido a orientação do professor em sala 

Tecnologias Utilizadas:
Linguagem C
Biblioteca stdio.h
Biblioteca stdlib.h
